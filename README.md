# Law Firm Intake AI Pipeline

AI-powered client intake automation for solo and small law firms.

## Endpoints
- GET /demo — Run demo with sample slip-and-fall case
- POST /intake/email — Process incoming client inquiry
- POST /intake/form — Process intake form submission
- POST /calendly/webhook — Handle consultation booking

## Deploy
Deployed on Render (free tier).
